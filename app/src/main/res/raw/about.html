Introduction

This app was designed to allow you to have a clear look of your timetable from ADE online services used by French Universities. It is supposed to work with any URL providing a valid *.ics file to direct download but this app was only tested using ADE services.

Open Source Libraries

com.journeyapps:zxing-android-embedded:3.6.0 : Used for QR code scanning used to save *.ics file download URL.

com.code-troopers.betterpickers:library:3.1.0 : Used for date picker.

org.mnode.ical4j:ical4j:1.0.7 : Used to parse *.ics file downloaded.

com.thebluealliance:spectrum:0.7.1 : Used to select a color from a palette in settings.

com.github.bumptech.glide:glide:3.7.0 : Used to select display tutorial images.

com.squareup.okhttp3:okhttp:4.3.0 : Used to select safely download your time schedule.
