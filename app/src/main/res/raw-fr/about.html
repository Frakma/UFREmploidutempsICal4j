Introduction

Cette application a été créée pour fournir une vision claire des emplois du temps fournis par les services en ligne ADE utilisés dans les Universités françaises. L'application est supposée fonctionner avec n'importe quel fichier *.ics disponible en téléchargement direct cependant, cette application n'a été testée qu'avec les services ADE.

Librairies Open Source

com.journeyapps:zxing-android-embedded:3.6.0 : Utilisé pour le scan de QR codes pour enregistrer l'URL de téléchargement du fichier *.ics.

com.code-troopers.betterpickers:library:3.1.0 : Utilisé pour le sélecteur de date.

org.mnode.ical4j:ical4j:1.0.7 : Utilisé pour analyser le fichier *.ics téléchargé.

com.thebluealliance:spectrum:0.7.1 : Utilisé pour la sélection de couleur dans les paramètres.

com.github.bumptech.glide:glide:3.7.0 : Utilisé pour la l'affichages des images du tutoriel.

com.squareup.okhttp3:okhttp:4.3.0 : Utilisé pour télécharger votre emploi du temps de manière sécurisée.
